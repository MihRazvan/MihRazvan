## Featured Engineering & Hackathon Wins

### [Autark](https://github.com/MihRazvan/Autark) *(1st Place Privacy - ETH Rome 2025)*
*A DevSecOps framework solving the "Single Dev Compromise" problem for self-sovereign frontend deployments.*
* **Immutable Releases:** Combines Safe multisig governance, ENS NameWrapper versioning, and IPFS (Storacha) into an atomic, verifiable release flow.
* **CLI Automation:** Built a robust Node.js/TypeScript CLI (`npm install autark`) to manage multi-party approvals before any UI code goes live.

### [Trust City Exchange](https://github.com/MihRazvan/agents) *(PL Genesis)*
*A live autonomous marketplace where specialized AI agents route, execute, and verify jobs using trust thresholds.*
* **Observable Autonomous Work:** Orchestrates a complex `discover -> plan -> execute -> verify` loop visualized through a live 3D React/Three.js client.
* **Verifiable Trust Layer:** Integrates real ERC-8004 identity and reputation receipts on Sepolia, ensuring work routing is backed by on-chain state, not just availability.

### [Cloakline](https://github.com/MihRazvan/cloakline) *(2nd Place Infra & Digital Rights - PL Genesis)*
*A drop-in privacy RPC for Ethereum that reduces wallet surveillance risk with a single RPC switch.*
* **Encrypted Logging:** Utilizes Zama fhEVM to keep operational logging encrypted, pseudonymous, and strictly write-only, removing raw wallet addresses from the chain.
* **Seamless Integration:** Acts as a sanitizing proxy layer between wallets and upstream providers without requiring users to change their transaction flows.

### [Bulwark](https://github.com/MihRazvan/bulwark_scroll) *(2nd Place DeFi - Scroll Open 2025)*
*An AI-driven, automated DeFi vault system optimizing yield strategies across lending and DEX protocols.*
* **Intelligent Execution:** Uses AI to analyze wallet holdings and execute smart contract batch transactions for optimal capital efficiency.
* **Risk Mitigation:** Features real-time liquidation monitoring and automated rebalancing to protect user capital from market volatility.
